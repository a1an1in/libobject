# libobject 用户手册

## 1. 简介

libobject 是一个基于 C 语言实现的面向对象编程基础库。它提供了一套完整的面向对象编程框架，包括对象系统、数据结构容器、网络通信库、数据库 ORM 框架、并发编程框架等模块。

### 1.1 功能特性

- **面向对象编程框架**：在 C 语言中实现类、对象、继承、多态等 OOP 特性
- **丰富的数据结构**：Vector、Map、List、Queue、Stack、String、Trie 等
- **数据库 ORM**：支持 MySQL，提供对象关系映射
- **网络通信**：HTTP 服务器/客户端、TCP/UDP、STUN/TURN、分布式总线
- **并发编程**：线程池、生产者-消费者、定时器、消息中心
- **脚本引擎**：FShell 命令行、Python 集成
- **跨平台**：支持 Linux、Windows、Android、macOS

## 2. 快速开始

### 2.1 环境要求

- CMake >= 3.10
- C 编译器（GCC、Clang 或 MSVC）
- MySQL 客户端库（可选，用于 ORM 模块）

### 2.2 编译安装

#### Linux 平台

```bash
# 克隆代码
git clone https://github.com/a1an1in/libobject.git
cd libobject

# 默认架构（x86_64）
./devops.sh build --platform=linux

# 指定架构
./devops.sh build --platform=linux --arch=x86_64
./devops.sh build --platform=linux --arch=arm
```

#### Android 平台

```bash
./devops.sh build --platform=android --arch=arm64-v8a
```

#### Windows 平台

```bash
# 原生编译
./devops.sh build --platform=windows

# 交叉编译
./devops.sh build --platform=windows --arch=x86_64
```

#### macOS 平台

```bash
./devops.sh build --platform=mac
```

### 2.3 部署

```bash
# Linux 部署到系统目录
./devops.sh release --platform=linux

# 部署到指定目录
./devops.sh release --platform=linux --prefix=/usr/local
```

## 3. 核心对象系统使用指南

### 3.1 定义类

使用 `DEFINE_CLASS` 宏定义类：

```c
#include <libobject/core/object.h>

// 定义结构体
struct MyClass_s {
    Obj obj;           // 继承自 Obj
    int value;         // 整型属性
    String *name;      // 字符串属性
};

// 定义类信息
DEFINE_CLASS(MyClass,
    Class_U32___Entry(value, NULL),
    Class_Str___Entry(name, NULL),
)
```

### 3.2 创建对象

```c
allocator_t *allocator = allocator_get_default_instance();

// 方式一：使用 object_new
MyClass *obj = object_new(allocator, "MyClass", "{value:42,name:\"test\"}");

// 方式二：使用 OBJECT_NEW 宏
MyClass *obj2 = OBJECT_NEW(allocator, MyClass, "{value:100,name:\"hello\"}");

// 销毁对象
object_destroy(obj);
```

### 3.3 设置和获取属性

```c
// 设置属性
obj->set(obj, "value", &42);
obj->set(obj, "name", "new_name");

// 获取属性
int *val = obj->get(obj, "value");
char *name = obj->get(obj, "name");
```

### 3.4 JSON 序列化

```c
// 对象转 JSON
char *json = obj->to_json(obj);
printf("JSON: %s\n", json);

// JSON 赋值
obj->assign(obj, "{value:200,name:\"updated\"}");
```

### 3.5 继承与多态

```c
// 定义子类
struct MyDerived_s {
    MyClass parent;
    int extra_field;
};

DEFINE_CLASS(MyDerived,
    Class_Obj___Entry(MyClass, parent),
    Class_U32___Entry(extra_field, NULL),
)

// 虚函数重写
static int __derived_to_json(Obj *obj) {
    // 自定义 JSON 序列化
    return 1;
}

// 在类信息中注册
// Init_Vfunc_Entry(5, MyDerived, to_json, __derived_to_json)
```

## 4. 数据结构使用指南

### 4.1 Vector（动态数组）

```c
// 创建 Vector
Vector *vec = object_new(allocator, "Vector", NULL);

// 添加元素
int value = 42;
vec->add(vec, &value);

// 获取元素
int *val = vec->get(vec, 0);

// 获取大小
int size = vec->size(vec);

// 遍历
for (int i = 0; i < vec->size(vec); i++) {
    int *v = vec->get(vec, i);
    printf("vec[%d] = %d\n", i, *v);
}

// JSON 序列化
char *json = vec->to_json(vec);
```

### 4.2 Map（关联容器）

```c
// 创建 Map
Map *map = object_new(allocator, "Map", NULL);

// 插入键值对
char *key = "key1";
int value = 100;
map->insert(map, key, strlen(key), &value, sizeof(int));

// 查找
int *val = map->get(map, "key1");

// JSON 序列化
char *json = map->to_json(map);
```

### 4.3 String（字符串）

```c
// 创建字符串
String *str = object_new(allocator, "String", "hello");

// 拼接
str->cat(str, " world");

// 格式化
str->format(str, "value=%d", 42);

// 获取 C 字符串
char *cstr = str->get_cstr(str);

// 获取长度
int len = str->length(str);

// 比较
int cmp = str->compare(str, "hello");
```

### 4.4 List（双向链表）

```c
// 创建 List
List *list = object_new(allocator, "List", NULL);

// 添加元素
list->add(list, &value);

// 遍历
Iterator *it = list->begin(list);
Iterator *end = list->end(list);
while (it->has_next(it)) {
    int *val = it->next(it);
}
```

## 5. 数据库 ORM 使用指南

### 5.1 初始化 ORM

```c
// 创建 ORM 实例
Orm *orm = object_new(allocator, "Orm", 
    "{db_type:\"mysql\",host:\"localhost\",port:3306,"
    "dbname:\"test\",user:\"root\",password:\"123456\"}");

// 获取数据库连接
Orm_Conn *conn = orm->get_conn(orm);
```

### 5.2 定义数据模型

```c
// 定义模型结构体
struct UserModel_s {
    Model model;
    int id;
    String *name;
    String *mobile;
};

// 定义模型类信息
DEFINE_CLASS(UserModel,
    Class_Obj___Entry(Model, model),
    Class_U32___Entry(id, NULL),
    Class_Str___Entry(name, NULL),
    Class_Str___Entry(mobile, NULL),
)

// 定义对应的表结构
struct UserTable_s {
    Table table;
};

DEFINE_CLASS(UserTable,
    Class_Obj___Entry(Table, table),
)
```

### 5.3 插入数据

```c
// 创建模型对象
UserModel *user = object_new(allocator, "UserModel", 
    "{name:\"user1\",mobile:\"13800138000\"}");

// 插入到数据库
conn->insert_model(conn, user);
```

### 5.4 查询数据

```c
// 查询模型
UserModel *result = NULL;
conn->query_model(conn, "UserModel", "WHERE id=1", &result);

// 查询表（返回多条记录）
Table *table = NULL;
conn->query_table(conn, "UserTable", "WHERE id>0", &table);

// 执行自定义 SQL 查询
char json_buf[4096];
conn->query(conn, json_buf, sizeof(json_buf), 
    "SELECT * FROM user_table WHERE id=%d", 1);

// 执行任意 SQL
conn->exec_sql(conn, "CREATE TABLE IF NOT EXISTS test (id INT)");
conn->exec_sql(conn, "INSERT INTO test VALUES(%d)", 42);
```

### 5.5 更新和删除

```c
// 更新数据
conn->update(conn, "UPDATE user_table SET name='new_name' WHERE id=1");

// 删除数据
conn->del(conn, "user_table", "WHERE id=1");
```

### 5.6 释放连接

```c
// 将连接归还到连接池
orm->add_conn(orm, conn);
```

## 9. 常见问题

### Q1: 编译时找不到头文件？

确保已正确设置 CMake 的包含路径：
```cmake
include_directories(${LIBOBJECT_PATH}/src/include)
link_directories(${LIBOBJECT_PATH}/sysroot/linux/x86_64/lib)
```

### Q2: 数据库连接失败？

检查 MySQL 服务是否运行，以及连接参数是否正确：
```c
Orm *orm = object_new(allocator, "Orm", 
    "{db_type:\"mysql\",host:\"127.0.0.1\",port:3306,"
    "dbname:\"test\",user:\"root\",password:\"your_password\"}");
```

### Q3: 如何调试？

使用内置的调试系统：
```c
// 设置调试级别
dbg_str(DBG_VIP, "important message");
dbg_str(DBG_WARN, "warning message");
dbg_str(DBG_ERROR, "error message");
```
