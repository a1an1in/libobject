## 三、软件功能与技术特点

### 3.1 核心功能

1. **面向对象编程框架**：在 C 语言中实现类定义、对象创建、继承、多态等 OOP 特性
2. **数据结构容器**：提供 Vector（动态数组）、Map（关联容器）、String（字符串）、List（链表）、Queue（队列）、Stack（栈）、Trie（前缀树）等
3. **数据库 ORM**：支持 MySQL 数据库，提供对象关系映射，支持 CRUD 操作和 JOIN 查询
4. **网络通信**：HTTP 服务器/客户端、TCP/UDP 通信、STUN/TURN NAT 穿透、分布式总线
5. **并发编程**：线程池、生产者-消费者模式、定时器、消息中心
6. **脚本引擎**：FShell 命令行解析引擎、Python 脚本集成
8. **加密编码**：MD5、SHA1、Base64、CRC32/CRC64
9. **归档压缩**：ZIP、Tar、7z、bzip2、gzip、SquashFS

### 3.2 技术特点

1. **创新的 C 语言 OOP 实现**：通过宏定义和注册机制，在 C 语言中实现了完整的面向对象编程范式
2. **运行时反射**：支持通过类名和属性名在运行时访问对象属性和方法
3. **JSON 原生支持**：所有对象支持 JSON 序列化和反序列化
4. **跨平台架构**：通过 OS 抽象层支持 Linux、Windows、Android、macOS
5. **插件化设计**：支持运行时动态加载类
